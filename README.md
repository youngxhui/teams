# 人员管理系统


# 所用技术



- 前端采用 Bootstrap jquery
    
- 后台使用 Struts2 Hibernate

- 语言 kotlin

# 功能

- [x] 登陆注册
- [x] 文档批量下载和上传
- [x] 地理位置签到
- [x] 公告/通知
- [x] 小组建立
- [x] 国际化
- [ ] 站内信
- [ ] 人员查询
